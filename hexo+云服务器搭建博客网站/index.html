    三、云服务器配置

    1、前期准备工作

    云服务器的购买

    可在下面的网站上注册登录后，点击“云服务器ECS ”进行购买

    • 阿里云主页 https://www.aliyun.com/
    • 腾讯云主页 https://cloud.tencent.com/
    • 我在腾讯云上领了一个月的轻量应用服务器尝尝鲜

    域名购买与注册

    • 新网主页https://www.xinnet.com/
    • 我在这上面买的域名，当时领了券，5元买了一年尝尝鲜。域名买好后先去备案。

    域名解析

    • 域名备案成功之后，需要设置域名解析，DNS解析设置里同时添加两条指向公网ip的主机记录：一条@记录，一条www记录。
    • 在浏览器输入www.fengmoqi.cn或者fengmoqi.cn，就可以打开我的博客网站了。它们都是基于http协议的，等同于http://www.fengmoqi.cn

    SSH工具连接服务器

    我用的是Windows电脑，推荐使用MobaXterm或者Xshell软件进行ssh连接。

    如果你用的是Mac电脑，推荐使用 Royal TSX 软件进行ssh连接。

    参考链接：在Mac上使用Royal TSX，替代 xshell 和 item2、SecureCRT，可以 SSH 也能 FTP

    连接好之后，接下来的操作都是在云服务器上操作。

    6、给 Git 用户配置 ssh key

    这样做的目的就是，以后由本地向服务器提交资源，就不需要再进行身份验证了。

    流程大致如下：

    • 先在本地的C:\Users\用户名.ssh目录生成公钥id_rsa.pub和私钥id_rsa；
    • 然后将本地的公钥文件id_rsa.pub上传到服务器端的 .ssh 文件夹；
    • 最后将公钥文件id_rsa.pub内容拷贝到 authorized_keys 文件中。

    我们在之前的步骤中已经配置过公钥到 Github仓库了，所以我们直接使用之前的公钥即可。

    另外，我们要注意在本地操作还是在服务器端操作；在服务器端的时候，是使用 root 用户还是使用 git 用户操作。
